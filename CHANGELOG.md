## [0.0.13] - 2026-07-06

### Changed

- 优化窄屏底部 dock，四个配置入口改为图标加文字上下排布，保留全部功能入口且减少信息拥挤。

## [0.0.12] - 2026-07-06

### Changed

- 将顶部和底部 dock、配置弹窗的磨砂玻璃进一步调整为更接近液态玻璃的高光、内描边和折射质感。
- 将底部配置弹窗动效改为从 dock 上沿展开的液态弹出效果，并增加弹窗与 dock 的连接层。
- 优化窄屏展示，压缩顶部模式 dock、主计时舞台、底部控制区和配置弹窗布局，减少遮挡和横向溢出。

## [0.0.11] - 2026-07-06

### Added

- 增加倒计时和正计时的“显示毫秒”选项，普通展示和全屏展示都会同步显示毫秒。
- 在顶部模式 dock 的应用标题旁展示版本号，点击可打开 GitHub 仓库。

### Changed

- 优化底部配置 dock 弹窗的玻璃质感、悬停反馈和弹出动效。
- 将主舞台、设置卡片和 dock 统一调整为更明显的磨砂玻璃视觉。
- 时钟模式默认不再展示标题，让时钟主视觉更干净。

## [0.0.10] - 2026-07-05

### Changed

- 优化模式切换时的标题语义；从“会议发言计时”等倒计时预设切到正计时或时钟时，会自动改为更通用的活动标题，避免三个模式都显示倒计时标题。
- 用户手动输入的自定义标题会在模式切换时保留，不被系统默认标题覆盖。

## [0.0.9] - 2026-07-05

### Added

- 增加顶部模式切换栏，支持在倒计时、正计时和时钟模式之间切换。
- 增加正计时模式，从 `00:00` 开始累计，支持开始、暂停、重置、快捷键和全屏大数字展示。
- 增加全屏时钟模式，以当前时间作为主视觉，并显示日期、星期和标题，适合课间、会议间歇和投屏待机。
- 为倒计时、正计时和时钟模式分别提供普通展示和全屏展示状态。

### Changed

- 将页面重构为顶部模式 dock、中间主内容和底部配置 dock 的结构。
- 将场景、基础、显示、提醒等具体配置改为从底部胶囊栏打开的浮窗。
- 页面背景改为简约渐变，并统一主舞台、浮窗、按钮和输入控件的圆角风格。
- 正计时和时钟模式下隐藏或禁用倒计时专属配置，减少无效设置干扰。

## [0.0.8] - 2026-07-05

### Changed

- 进一步压缩提醒设置区域高度，减少提醒卡片、输入框和快捷节点的垂直占用。
- 移除提醒卡片标题中的重复时间副标题，让展开后的提醒列表更紧凑。

## [0.0.7] - 2026-07-05

### Added

- 增加屏幕常亮设置，默认通过 Screen Wake Lock API 尝试保持投屏设备屏幕常亮，并在“显示与计时”中展示授权状态。
- 增加手动“启用常亮”入口，浏览器首次进入页面或首次交互后可触发屏幕常亮权限请求。

### Changed

- 页面恢复可见时会自动重新申请屏幕常亮，关闭屏幕常亮设置或离开页面时会释放 Wake Lock。

## [0.0.6] - 2026-07-05

### Added

- 增加可开关的全屏进度条，大屏模式可显示类似 iPhone 横屏 StandBy 计时器的左侧经过时间进度区域。

### Changed

- 统一计时数字字体和字重，全屏主数字、右上角真实时间、已进行/剩余时长等数字使用一致的圆润数字样式。
- 全屏进度模式下标题和右下角控制按钮会在闲置后自动隐藏，鼠标、触摸或键盘操作后恢复显示。
- 优化全屏进度模式的数字垂直居中和文字留白，减少大字号数字顶部被裁切的视觉问题。

### Fixed

- 修正横屏低高度下全屏外层 padding 被断点覆盖导致的页面滚动、数字偏移和右上角真实时间外框不匹配。

## [0.0.5] - 2026-07-05

### Changed

- 将首页重构为 Notion 风主控台布局，突出主计时区并降低页面装饰和设置干扰。
- 移除设置抽屉，改为固定右侧栏 accordion，场景、基础、显示与计时、提醒可在右栏内单项展开编辑。
- 重做提醒编辑体验，增加常用节点快捷添加，并将分秒输入改为紧凑的内联单位样式。
- 优化桌面、宽屏、矮屏和移动端基础响应式布局，减少横向溢出和控件拥挤。

### Fixed

- 修正大屏超时后底部文案，超时状态显示“已超时 +00:xx”而不是“剩余 +00:xx”。

## [0.0.4] - 2026-07-02

### Changed

- 将普通页面调整为左侧计时主控台、右侧轻量设置检查器的排布，开始、重置、全屏和声音控制整合进主计时面板。
- 将场景模式从双列卡片改为紧凑列表，避免预设落单并降低设置区视觉干扰。
- 优化桌面和移动端响应式布局，减少控制区割裂感并提升小屏按钮可用性。

### Fixed

- 将 Service Worker 缓存策略改为在线优先、离线兜底，避免发布新版后旧缓存 HTML 引用过期构建资源导致空白页。

## [0.0.3] - 2026-07-02

### Added

- 增加演讲模式、课堂展示、答辩模式、会议发言和自由计时等场景预设，可一次套用时长、提醒节点、超时和全屏显示设置。
- 增加“全屏显示真实时间”选项，大屏模式可在右上角弱显示当前时间。

### Changed

- 将普通页面重设计为实时预览小窗和分组配置工作台，配置区按场景、基础、显示与计时、提醒拆分。
- 默认标题改为“现场计时”，减少对单一赛课场景的绑定。
- 优化预览小窗视觉，移除顶部色条并改为更克制的浅色卡片风格。
- 大屏模式新增标题展示，并保留倒计时作为主视觉。

## [0.0.2] - 2026-07-02

### Added

- 增加已用时展示；普通模式展示剩余、已用时和总时长，大屏模式展示主时间、已进行时长和剩余时长。
- 增加更醒目的提醒阶段、最后阶段和超时视觉提示，超时主数字以 `+00:00` 形式展示。
- 增加提醒节点有效性提示，提醒时间大于等于总时长时会说明该节点不会触发。
- 增加提醒节点 `0分0秒` 状态，设置为 0 秒时自动视为未启用。

### Changed

- 优化大屏展示模式，仅保留主时间、已进行时长和剩余时长，提醒改为整屏背景色变化，控制区改为右下角图标组。
- 运行中重置改为二次确认，降低现场误操作风险。

## [0.0.1] - 2026-07-01

### Added

- 初始化 TimerDisplay 中文网页倒计时器。
- 支持标题、常用预设、自定义小时/分钟/秒和提醒节点配置。
- 支持开始、暂停、继续、重置、全屏大屏模式和快捷键操作。
- 支持剩余 3 分钟、剩余 1 分钟、超时等阶段颜色提示。
- 支持提示音、试听提示音、超时后继续正计时和本地设置保存。
- 支持静态构建和已访问后的离线可用增强。
- 发布生产环境：[https://timer-display.vercel.app](https://timer-display.vercel.app)。
